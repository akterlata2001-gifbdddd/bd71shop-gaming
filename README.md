# 🎮 Bd71shop.com — Digital Gaming Gift Cards & Top-Up Store

A modern, neon-themed multi-page e-commerce frontend for **BD71SHOP** — Bangladesh's premier digital online global game store. Built with Next.js 16, TypeScript, Tailwind CSS 4, and shadcn/ui.

> Reference site: [bd71shop.com](https://bd71shop.com)

---

## ✨ Features

### 📄 15 Pages (Single-Page App with client-side routing)
- **Home** — Hero, trust badges, deals of the day, top categories, popular gift cards, best seller gaming top-up, blog preview, newsletter
- **Shop** — Sidebar category filters, search, sort options, grid/list view
- **Product Detail** — Image gallery, quantity selector, tabs (Description / Reviews / Delivery & Returns), related products
- **Cart** — Quantity controls, coupon field, order summary, recommended products
- **Checkout** — Contact info, game account details, payment method (bKash / Nagad / COD / Card), success screen
- **About** — Brand story, stats, services, why choose us, CTA
- **Contact** — Contact form (with subject dropdown), contact methods, support banner
- **Blog List** — Featured post, search, category filters, post grid
- **Blog Single** — Full article with TOC, share buttons, related posts
- **Privacy Policy / DMCA / Disclaimer** — Legal pages with TOC
- **404 Not Found** — Gaming-themed fallback
- **Account** — Login/Register toggle + dashboard (orders, wishlist, settings)
- **Order Tracking** — Order number search with delivery timeline

### 🛒 Interactive Features
- Slide-out Cart Drawer (auto-opens on add)
- Persistent cart state (zustand)
- Client-side routing with smooth transitions
- Live filtering & sorting on Shop page
- Add-to-cart feedback (button morphs to "Added ✓")
- Newsletter subscription with success animation

### 🎨 Design System
- **Gaming dark theme** — deep indigo background with neon purple, electric cyan, hot magenta accents
- **Fonts** — Orbitron (display) + Outfit (body) + JetBrains Mono
- **Custom animations** — floating cards, pulsing glow, gradient shifts, grid background
- **Mobile-first responsive**
- **Sticky footer**
